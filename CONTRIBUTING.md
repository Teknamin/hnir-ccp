# Contributing to HNIR-CCP

HNIR-CCP is a completed research artifact at v1.0.

## Reporting Issues

If you find bugs or have questions about the codebase, please open an issue.

## Development Setup

```bash
git clone https://github.com/Teknamin/hnir-ccp.git
cd hnir-ccp
pip install -e ".[dev]"
pytest
```
